## What this is

TARP is a reference wiki, not a guide. Pages describe how things work — movement physics, weapon behavior, gametype rules — without coaching language or opinion. Facts are checked against game source code or first-hand testing before they're published.

The project is built on [Quartz 5](https://quartz.jzhao.xyz/) and deployed via GitHub Actions on every content update.

## Contributing

This wiki is curated — pages go through a review step before they're published to the site. If you spot something wrong or missing, open an issue.
