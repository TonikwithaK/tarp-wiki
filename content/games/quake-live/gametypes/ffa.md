# Free For All

Verified from `ffa.factories` (quakelive-server-standards).

## Format

Free-for-all deathmatch. Every player is an opponent. Most frags by time limit wins. Continuous respawn.

## Mode Settings

| cvar | Value | Notes |
|---|---|---|
| `timelimit` | 15 | Minutes |
| `g_overtime` | 0 | No overtime |
| Score limit | none | No score limit by default |

## Spawn

Machine Gun + 125 HP. All other weapons and items acquired from map pickups.

Item pickups present. Item respawn timers apply.
