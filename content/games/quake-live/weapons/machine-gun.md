# Machine Gun

The Machine Gun (MG) fires rapid-fire hitscan shots dealing 5-7 damage per shot. Players spawn with the Machine Gun and 100 ammo in all standard gametypes. It is the default starting weapon.

## Stats

| Stat | Value |
|---|---|
| Damage per shot | 5-7 |
| Splash damage | None |
| Self-damage | None |
| Starting ammo | 100 |

## Mechanics

High fire rate, low per-shot damage. Hitscan with no splash. Ammo is abundant: 100 on spawn plus pickups.
