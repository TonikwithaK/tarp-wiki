# Button Commands

Button commands are held-key inputs used with `bind <key> <command>` in console or autoexec.cfg. All use the `+` prefix.

| Command | Description |
|---------|-------------|
| `+attack` | Fire weapon |
| `+button0` | Fire weapon (alternate bind) |
| `+button1` | Display chat bubble |
| `+button2` | Use holdable item |
| `+button3` | Gesture |
| `+button5` | Yes / I vote yes (Team Arena models only) |
| `+button6` | No / I vote no (Team Arena models only) |
| `+button7` | Attack signal: I am on offense (TA models only) |
| `+button8` | Defend signal: I am defending (TA models only) |
| `+button9` | Patrol signal: I am patrolling (TA models only) |
| `+button10` | Follow signal: Follow me (TA models only) |
| `+chat` | Display chat window |
| `+forward` | Move forwards |
| `+back` | Move backwards |
| `+moveleft` | Strafe left |
| `+moveright` | Strafe right |
| `+moveup` | Jump / swim up |
| `+movedown` | Crouch / swim down |
| `+left` | Turn left |
| `+right` | Turn right |
| `+lookup` | Look up |
| `+lookdown` | Look down |
| `+mlook` | Mouse look |
| `+speed` | Run/Walk toggle |
| `+strafe` | Strafe toggle |
| `+zoom` | Zoom toggle |
| `+scores` | Display scoreboard |
| `+acc` | Display in-game weapon accuracy panel |
