Base Q3 ships with 4 gametypes. Team Arena added additional modes not covered here.

| Gametype | g_gametype | Description |
|---|---|---|
| Free For All | 0 | Every player for themselves. Highest frag count wins. Up to 16 players. |
| Tournament | 1 | 1v1 duel. Spectators queue and rotate in after each match ends. |
| Team Deathmatch | 3 | Red vs Blue. Highest combined team frag total wins. Standard format is 4v4. Machine Gun deals 5 damage instead of 7. |
| Capture the Flag | 4 | Team-based. Steal the enemy flag and return it to your base while defending your own. Uses 5 symmetrical CTF maps. |

Spawn weapons in all modes: Gauntlet + Machine Gun.

## Map Counts (base game + patches)

- FFA/DM maps: 30 total (26 base, 4 added via updates)
- CTF maps: 5 total (4 base, 1 added via update)
- Pro maps added via updates: PRO-Q3DM6, PRO-Q3DM13, PRO-Q3TOURNEY2, PRO-Q3TOURNEY4
