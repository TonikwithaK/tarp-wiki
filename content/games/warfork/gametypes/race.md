# Race

Race is a movement mode. Players complete a course in the fastest time. There is no combat and no damage between players.

## Rules

- Complete the course as fast as possible.
- Best times are recorded and tracked per map.
- Respawning returns the player to the course start or the last checkpoint.
- Maps follow the naming convention: `wfrace[id]`.
- Plasmaclimbing is required on many routes.

## Numbers

| Parameter | Value |
|---|---|
| Score limit | none |
| Time limit | none |
| Item respawn | 1 second |
| Infinite ammo | yes |
| Team-based | no |
| Spawn radius | 0 (exact spawn point) |
